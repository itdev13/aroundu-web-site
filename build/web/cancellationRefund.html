Last Updated: 30/06/2025

Event Cancellation
In the event that a scheduled event is canceled by the organizer, AroundU guarantees a full refund of the ticket price without any deductions.

The refund will be processed automatically within 5-7 business days and will be credited to the original payment method used for the transaction.

User-Initiated Refund Requests

If the event host (House/User) allows refunds, this option will be displayed on the AroundU app. Users can initiate a refund request up to 2 days before the event if permitted by the event host.

  • If a refund request is approved, the amount will be credited back to the original payment method within the standard processing time.
  • If a refund request is denied by the event host, no refund will be issued.
  • AroundU does not bear responsibility for refunds in cases where the event host has explicitly disallowed them. Any disputes must be resolved between the user and the event host.
